# Enterprise AI Assistant

An enterprise-grade AI Assistant built using

- Streamlit
- LangChain
- Groq
- FAISS
- HuggingFace Embeddings

## Features

- AI Chat
- RAG
- PDF Loader
- DOCX Loader
- TXT Loader
- Vector Search
- Voice Agent (Coming Soon)

## Installation

```bash
git clone <repo>

cd enterprise-ai-assistant

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

Create

```
.env
```

Add

```
GROQ_API_KEY=YOUR_KEY
```

Run

```bash
streamlit run app.py
```
